Tutorial rápido: criar usuário e senha de funcionário

Este passo a passo mostra como cadastrar um novo acesso para funcionário usando a tela de Configurações no menu.

Abrir Gestão de Equipe Abrir Novo Funcionário

Passo a passo

  1. No sistema, abra o Menu principal.
  2. Clique em Configurações.
  3. Entre na área Funcionários / Usuários (Gestão de Equipe).
  4. Clique em Novo usuário / Adicionar funcionário.
  5. Preencha os dados básicos do funcionário (nome, e-mail e/ou telefone).
  6. No campo de acesso, defina o usuário/login.
  7. Crie uma senha inicial (de preferência com letras, números e símbolo).
  8. Selecione o perfil/permissão do funcionário (ex.: atendente, mecânico, gerente).
  9. Clique em Salvar para finalizar o cadastro.
  10. Peça para o funcionário entrar no sistema com o novo login e trocar a senha no primeiro acesso, se essa opção existir.
Dica: antes de salvar, revise o perfil de permissão para evitar acesso indevido a funções administrativas.

Arquivo de teste simples para treinamento interno. Se seu sistema estiver em outro endereço, troque http://localhost:3000 pelos seus links.
