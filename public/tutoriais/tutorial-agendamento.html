Agendamento

Como criar agendamentos, enviar o link ao cliente, usar os alertas em tempo real e acompanhar tudo pelo painel.

1

O que é o agendamento

O módulo de agendamento permite marcar um dia e horário com um cliente — seja para uma avaliação inicial do veículo (sem OS aberta), seja para um retorno de uma OS que já existe.

Ao criar o agendamento, o sistema gera um link exclusivo que você envia ao cliente pelo WhatsApp. Pelo link, o cliente confirma a presença — e você vê isso em tempo real no painel.

Quando a hora chegar, um alerta automático aparece no topo do sistema, com som e notificação, para que você não perca nenhum atendimento.

2

O painel de agendamentos

Acesse pelo menu lateral em Agendamentos. A tela tem dois elementos principais:

Calendário do mês

No topo, um calendário mostra todos os dias do mês. Clique em qualquer dia para ver a lista de agendamentos daquele dia. O sistema destaca os dias com mais agendamentos para facilitar a visualização da agenda.

Mapa de calor de ocupação por hora

Abaixo do calendário, uma linha de horários (das 7h às 18h) mostra o quanto cada hora do dia selecionado está ocupada:

Use o mapa de calor para decidir o melhor horário ao criar um novo agendamento — evitando concentrar tudo na mesma hora.

Lista de agendamentos do dia

Abaixo, aparecem todos os agendamentos do dia em ordem de horário. Para cada um você vê: nome do cliente, veículo, horário, duração e status (ícone colorido). Os botões de ação ficam ao lado de cada agendamento.

3

Criar um agendamento

Clique em + Novo Agendamento no topo da tela. O formulário pede primeiro o tipo do agendamento — e o restante muda conforme o tipo escolhido:

Selecione o tipo para ver as instruções:

Use quando o cliente vai trazer o veículo pela primeira vez, sem OS aberta. O objetivo é avaliar o problema e decidir o que fazer.

Campos do formulário:

  • Cliente — campo de busca opcional. Digite o nome para buscar um cliente já cadastrado. Se for um cliente novo, deixe em branco e anote o nome na descrição.
  • Veículo — campo de texto livre. Escreva a placa ou modelo (ex: "Gol 2018 – ABC1234"). Não é vinculado ao cadastro.
  • Data e hora — selecione o dia e a hora do atendimento. O sistema não permite marcar no passado.
  • Duração — tempo estimado para o atendimento: 15min, 30min, 1h, 2h, 4h ou dia inteiro. Isso afeta o mapa de calor de ocupação.
  • Descrição — escreva o motivo da visita (ex: "Problema no alternador"). Obrigatório se não houver cliente selecionado.
💡 Ao criar este agendamento, nenhuma OS é aberta ainda. Quando o cliente chegar, você abre a OS normalmente pelo menu de Atendimento — e pode vincular a OS ao agendamento depois, se quiser.

Use quando um cliente já tem uma OS aberta e precisa retornar — por exemplo, para aprovação de orçamento, buscar o veículo, ou continuar um serviço.

Campos do formulário:

  • Buscar OS — campo de busca. Digite o número da OS, o nome do cliente ou a placa. Uma lista aparece abaixo; clique na OS desejada.
  • Após selecionar a OS, o sistema exibe automaticamente: nome do cliente, veículo e descrição herdados da OS.
  • Data e hora — selecione quando o cliente vai retornar.
  • Duração — tempo estimado do retorno.
📋 O agendamento fica vinculado à OS. Quando o cliente acessar o link pelo portal, ele verá tanto o agendamento quanto a situação atual da OS (status, orçamento para aprovar, etc.).
4

Enviar o link ao cliente

Depois de criar o agendamento, ele aparece na lista do dia. Clique no ícone do WhatsApp ao lado do agendamento para enviar o link de confirmação ao cliente.

O sistema abre o WhatsApp Web com uma mensagem já pronta, que inclui:

  • Nome do cliente.
  • Data e hora do agendamento.
  • Um link exclusivo para o portal do cliente.
💡 Se o cliente não tiver WhatsApp cadastrado, o sistema pede que você informe o número antes de abrir o WhatsApp. O número digitado é salvo para uso futuro.
🔗 O link gerado é exclusivo para esse agendamento. Não expira e pode ser reenviado quantas vezes precisar — por exemplo, se o cliente perder o link.
5

O que o cliente vê no portal

Ao abrir o link, o cliente chega numa tela com as informações do agendamento e um botão para confirmar a presença. A aparência muda conforme o status:

Sexta-feira, 25 de abril — 10h00 • Avaliação do alternador
Confirmar Presença
✅ Presença confirmada

❌ Agendamento cancelado

Atendimento concluído.
📋 Se o agendamento for do tipo "Já tem OS", o cliente vê também a situação da OS no mesmo portal — linha do tempo de status, fotos, orçamento para aprovar e documentos fiscais. Ele acessa tudo por um único link.
6

Alertas em tempo real

O sistema monitora os agendamentos do dia automaticamente. Quando um atendimento está prestes a começar, um banner de alerta aparece no topo do painel, acompanhado de um som e uma notificação do navegador.

⏰ Agendamento chegando
João Silva — Gol 2018 (ABC1234)
Previsto para 10h00 • Avaliação do alternador • Confirmado ✓
Avisar +15min Não compareceu Reagendar Abrir OS

O alerta aparece 15 minutos antes do horário marcado — e permanece enquanto o agendamento estiver nos status "agendado" ou "confirmado". No alerta, você pode:

  • Avisar +15min — "dorme" o alerta por 15 minutos se você já sabe que o cliente está a caminho ou está esperando. O alerta volta depois.
  • Não compareceu — marca o agendamento como "Não Compareceu" e remove o alerta. O cliente verá isso no portal.
  • Reagendar — abre a tela para escolher nova data e hora, sem precisar entrar no painel de agendamentos.
  • Abrir OS — cria uma nova OS já vinculada ao agendamento, abrindo diretamente a tela de novo atendimento. Ideal para avaliações que vão gerar serviço.
🔔 Para receber as notificações do navegador, permita quando o sistema pedir. Se negou por engano, acesse as configurações do seu navegador e autorize o site manualmente.
💡 O alerta aparece em qualquer tela do painel — não só na tela de agendamentos. Você não precisa ficar olhando o relógio.
7

Ações disponíveis no agendamento

Na lista de agendamentos do dia, cada item tem ícones de ação ao lado. Role a linha até o final se estiver no celular:

  • WhatsApp — envia ou reenvia o link de confirmação para o cliente.
  • Reagendar (relógio) — abre uma tela para escolher nova data e hora. O status é resetado para "agendado" automaticamente após reagendar.
  • Concluir (check) — marca o agendamento como concluído, sem abrir uma OS. Use quando o atendimento aconteceu mas não gerou serviço (ex: cliente veio buscar peça, avaliação rápida sem custo).
  • Cancelar (X) — cancela o agendamento. O cliente verá "Agendamento Cancelado" no portal, com um botão para chamar a oficina pelo WhatsApp.
📋 Agendamentos cancelados ficam registrados no histórico — eles não somem completamente. Mas são removidos da lista do dia para não poluir a visualização.
8

Reagendar um atendimento

Para mover um agendamento para outra data ou hora, clique no ícone de relógio ao lado do agendamento. Um formulário aparece com a data e hora atuais já preenchidas.

  1. Escolha a nova data.
  2. Escolha o novo horário.
  3. Clique em Salvar.

Após reagendar, o status volta para "agendado" — mesmo que o cliente já tivesse confirmado. Reenvie o link pelo WhatsApp para que o cliente veja o novo horário.

💡 Se o cliente pediu para reagendar pelo WhatsApp, faça a mudança no painel e reenvie o link. O cliente verá a nova data e hora ao abrir o mesmo link de antes — não precisa gerar um link novo.
9

Status dos agendamentos

Cada agendamento passa por um ou mais dos seguintes status:

🕐
Agendado
Status inicial. O agendamento foi criado e o link foi enviado, mas o cliente ainda não confirmou.
✅
Confirmado
O cliente abriu o link e clicou em "Confirmar Presença". Um checkmark verde aparece no painel.
🏁
Concluído
Atendimento finalizado sem geração de OS. Marcado manualmente pelo mecânico.
😔
Não compareceu
O cliente não apareceu. Marcado pelo mecânico no alerta ou na lista. O cliente vê isso no portal.
❌
Cancelado
Agendamento cancelado pelo mecânico. O cliente vê um aviso no portal com botão para ligar ou chamar no WhatsApp.
10

Fluxo completo de uma visita

Para facilitar, veja como funciona o ciclo completo de um agendamento de avaliação do início ao fim:

1
Mecânico cria o agendamento (tipo Avaliação), escolhe data, hora e escreve o motivo da visita.
2
Mecânico clica no ícone do WhatsApp e envia a mensagem com o link para o cliente.
3
Cliente abre o link, vê o agendamento e clica em "Confirmar Presença". O painel do mecânico atualiza para status "Confirmado" com checkmark verde.
4
Quando faltam 15 minutos: alerta aparece no topo do sistema com som. Mecânico vê nome, veículo e horário do cliente.
5
Cliente chega. Mecânico clica em "Abrir OS" no alerta ou abre um novo atendimento normalmente. O serviço começa.
6
Se o serviço precisar de retorno: mecânico cria um novo agendamento do tipo "Já tem OS", vinculado à OS aberta. O cliente usa o mesmo link da OS para confirmar o retorno.
